# Content Writer

Content Writers are AI agents specialized in creating specific types of content. They can be customized with different writing styles, expertise levels, and domain knowledge.

## Capabilities
- Generate structured content based on prompts
- Adapt writing style to match brand voice
- Follow knowledge base guidelines
- Create content in multiple formats
- Maintain consistency across documents

## Configuration
- Writing style (technical, casual, formal)
- Domain expertise (technology, marketing, etc.)
- Content type specialization
- Tone and voice preferences
